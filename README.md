# tweet-chat

This is a really simple proyect made just for fun.
It has been done using:
 - WebSockets
 - Google Maps
 - Twitter Bootstrap
 - Jquery
 - Node.js

### Instalation

```
$ git clone https://github.com/termosfera/tweet-chat.git
$ npm install && bower install
```

### Run the server

```
$ node server.js
```
